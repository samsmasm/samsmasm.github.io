# Team Work

Short check-in activities for the start or end of class.

These activities are about staying connected as a group: sharing how things are going, noticing patterns, and supporting each other through the process. They are not about planning your IA together - that is your own work. Each takes about 5–10 minutes. Use paper, a whiteboard, or a shared digital board, and take a quick photo or screenshot to keep a record.

---

## Feeling check-ins - How are you doing right now?

### [TAB] One Word

*A quick pulse-check. Good for the start of class.*

Go around the group. Each person says (or writes) **one word** that describes how their IA work is going right now.

```
  [ focused ]   [ stuck ]   [ energised ]   [ overwhelmed ]   [ okay ]
```

Once everyone has shared, pause for a moment. Is there a pattern? Does anyone want to say a little more about their word?

### [TAB] Happiness Radar

*Maps how people are feeling across different parts of the IA. Good for identifying where support is needed.*

Draw a grid on paper or a shared board:

```
              😊 Happy    😐 Okay    😞 Struggling
            +-----------+-----------+-----------+
  Research  |           |           |           |
            +-----------+-----------+-----------+
  Writing   |           |           |           |
            +-----------+-----------+-----------+
  Time/pace |           |           |           |
            +-----------+-----------+-----------+
```

Each person puts a mark (or sticky note) in the cell that matches how they feel about each area. Look at the results together - what stands out?

*You can change the row labels to fit where you are in the IA process.*

### [TAB] Hopes and Concerns

*Surfaces what's on people's minds before a new phase of work. Good before a deadline or a new section.*

Split your paper or board into two columns:

```
  +----------------------+----------------------+
  |        HOPES         |      CONCERNS        |
  |                      |                      |
  |  What are you        |  What are you        |
  |  hoping for in       |  worried about in    |
  |  the next phase?     |  the next phase?     |
  |                      |                      |
  +----------------------+----------------------+
```

Each person writes 1–2 sticky notes for each column. Share and discuss. Are there common concerns you could help each other with?

## Reflection - Looking back at what's happened

### [TAB] Three Sights

*Reflects on what's hard, what's working, and what comes next. Good for a mid-process check-in.*

Three columns, three questions:

```
  +------------------+------------------+------------------+
  |   PLAIN SIGHT    |     INSIGHT      |    FORESIGHT     |
  |                  |                  |                  |
  |  What's hurting? |  What's working? |  What's next?    |
  |                  |                  |                  |
  +------------------+------------------+------------------+
```

Each person writes one note per column. Share them, then spend a moment on the "What's next?" column - what do you each actually plan to do before the next class?

### [TAB] Hot-Air Balloon

*Looks back at what helped and what slowed you down, then looks forward at risks. Good for the middle of the IA process.*

Draw a balloon diagram:

```
             ~~~~~ hot air ~~~~~
            (things pushing you UP)
                    | |
                  __|_|__
                 /       \
                | BALLOON |
                 \_______/
                    | |
             -------| |-------
            (things pulling you DOWN)
                    | |
             [sandbags / friction]

        <-- storm ahead -->  <-- sunny day -->
        (risks coming up)    (ways to avoid them)
```

Go through four areas:

- **Hot air** - what's been helping you move forward?
- **Sandbags** - what's been slowing you down?
- **Storm ahead** - what risks do you see coming?
- **Sunny day** - what can you do to steer around those risks?

Share your notes and compare. You'll often find you're facing the same storms.

### [TAB] Hero's Journey

*A reflective storytelling activity. Good near the end of the IA, or at a point when morale needs a boost.*

Four areas on a shared canvas:

```
  +-------------------+-------------------+
  |    THE HERO       |    THE GUIDE      |
  |  (you, on your    |  (people, tools,  |
  |   IA journey)     |   feedback that   |
  |                   |   helped you)     |
  +-------------------+-------------------+
  |    THE CAVERN     |   THE TREASURE    |
  |  (the challenges  |  (what you're     |
  |   ahead or        |   working toward  |
  |   already faced)  |   or achieved)    |
  +-------------------+-------------------+
```

Each person adds notes to each area for their own journey. Share them. It's often easier to be honest about challenges when the framing feels a bit like a story.

## Clarifying your thinking - Working through uncertainty or decisions

### [TAB] Sphere of Influence

*Helps separate what you can act on from what you can only worry about. Good when the group feels stuck or stressed.*

Draw three concentric circles and label them:

```
        +---------------------------------------+
        |            CONCERN                    |
        |     (worries you can't change)        |
        |   +---------------------------+       |
        |   |        INFLUENCE          |       |
        |   |  (you can nudge this)     |       |
        |   |   +-----------------+    |       |
        |   |   |    CONTROL      |    |       |
        |   |   | (fully yours)   |    |       |
        |   |   +-----------------+    |       |
        |   +---------------------------+       |
        +---------------------------------------+
```

Each person writes down the things that are stressing them about the IA and places them in the right ring. Talk through anything in the outer ring - can the group help move any of those items inward?

### [TAB] 3Ds: Desire, Data, Doubt

*Good for when you're trying to make a decision - like narrowing your research question or choosing an approach.*

Three columns:

```
  +--------------+--------------+--------------+
  |   DESIRE     |     DATA     |    DOUBT     |
  |              |              |              |
  | What are you | What do you  | What are you |
  | trying to    | actually     | unsure about |
  | achieve?     | know so far? | or need to   |
  |              |              | find out?    |
  +--------------+--------------+--------------+
```

Each person fills in all three for their own IA situation. Share your "Doubt" column with the group - sometimes others have already worked through the same uncertainty.

### [TAB] Success Criteria

*Defines what "done well" looks like for a specific goal. Good before tackling a tricky section.*

Set up a grid:

```
  +------------------+-------------------+-------------------+
  |  TARGET OUTCOME  |   SUCCESS IF...   |   FAILURE IF...   |
  +------------------+-------------------+-------------------+
  |                  |                   |                   |
  |                  |                   |                   |
  +------------------+-------------------+-------------------+
```

Each person identifies one concrete goal for the next phase of their IA, then fills in what success and failure look like for that goal. Share and compare - does everyone's "success" look reasonable? Does anyone's look too vague or too ambitious?

## Planning ahead - Organising what comes next

### [TAB] Now, Next, Later

*Clarifies priorities without getting into rigid deadlines. Good for planning the road ahead.*

Three columns on your board:

```
  +--------------+--------------+--------------+
  |     NOW      |     NEXT     |    LATER     |
  |              |              |              |
  | What you're  | What follows | What's still |
  | working on   | after that   | further off  |
  | right now    |              |              |
  +--------------+--------------+--------------+
```

Each person fills in their own column set (or you can do it as a group, comparing side by side). Useful for noticing if someone is stuck at the same "Now" they were in last week.

### [TAB] GIST

*Good for breaking down a goal into concrete steps. Use this when someone knows where they want to go but not how to get there.*

Four levels, from big to small:

```
  GOAL
  └── What's the big objective? (e.g. "complete my analysis section")
      │
      IDEA
      └── What's your approach? (e.g. "use two companies for comparison")
          │
          STEPS
          └── What are the sub-tasks? (e.g. "find data, draft comparison, add theory")
              │
              TASKS
              └── What will you do TODAY? (e.g. "find revenue figures for Company A")
```

Each person maps out their own GIST. Share your "Tasks" row with the group - this is your commitment for next class.

### [TAB] Three Horizons

*Zooms out to see short-, medium-, and long-term priorities at once. Good mid-IA when things feel urgent but you're losing the bigger picture.*

Three columns with your own time labels:

```
  +------------------+------------------+------------------+
  |    HORIZON 1     |    HORIZON 2     |    HORIZON 3     |
  |   (this week)    |   (next 2 weeks) |  (rest of IA)   |
  |                  |                  |                  |
  |  What am I       |  What comes      |  What do I need  |
  |  focused on now? |  after that?     |  to not forget?  |
  +------------------+------------------+------------------+
```

Each person fills in their own three horizons. Compare: is anyone overloaded in H1 but ignoring H3?
