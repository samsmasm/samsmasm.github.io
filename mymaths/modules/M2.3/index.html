Ordinary Differential Equations

Overview

The language of dynamics in physics, biology, economics, and engineering. Exact solution methods, qualitative analysis, phase plane techniques, and numerical methods.

Why This Matters

ODEs model everything that changes over time. Systems of ODEs via matrix methods connect directly to linear algebra. Phase plane analysis is the geometric lens on dynamics — essential for stochastic calculus later.

Learning Outcomes

  1. 1Solve first-order ODEs (separable, linear, Bernoulli, exact).
  2. 2Solve second-order linear ODEs with constant coefficients (homogeneous and inhomogeneous).
  3. 3Use Laplace transforms to convert and solve initial value problems.
  4. 4Solve systems of linear ODEs via matrix methods and matrix exponentials.
  5. 5Construct phase portraits; classify critical points by stability.
  6. 6Model real-world problems as ODEs and interpret solutions.
  7. 7Apply Euler's method and Runge-Kutta numerically in Python.

⚡ Milestone Checkpoint

Solve a damped harmonic oscillator completely. Sketch the phase portrait for a nonlinear system and classify all critical points.

AI Integration

Stability of gradient flows; neural ODEs; dynamical systems perspective on learning algorithms.
