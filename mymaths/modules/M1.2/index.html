Linear Algebra

Overview

Linear algebra is the language of AI, data science, physics, and virtually all quantitative fields. Follows Strang's Four Fundamental Subspaces approach, building toward SVD and eigenanalysis.

Why This Matters

Nearly every advanced module uses matrices, vector spaces, or eigenanalysis. This is the most practically useful single module in the programme.

Learning Outcomes

  1. 1Solve systems of linear equations via Gaussian elimination; compute LU decomposition.
  2. 2Define and identify vector spaces, subspaces, bases, and dimension.
  3. 3Find the four fundamental subspaces (column space, null space, row space, left null space) of a matrix.
  4. 4Apply the Gram-Schmidt process and construct orthogonal projections.
  5. 5Solve least-squares problems for overdetermined systems.
  6. 6Compute eigenvalues and eigenvectors; diagonalise matrices.
  7. 7Understand and compute the Singular Value Decomposition (SVD).
  8. 8Use linear algebra for geometric transformations, Markov chains, and network analysis.

⚡ Milestone Checkpoint

Given a matrix A, find all four fundamental subspaces and verify the rank-nullity theorem. Compute eigenvalues and eigenvectors. Compute the SVD of a 2×2 matrix by hand.

AI Integration

Ask for geometric interpretations of each subspace. Ask for ML applications of each concept (SVD → image compression, PCA, recommendation systems).
