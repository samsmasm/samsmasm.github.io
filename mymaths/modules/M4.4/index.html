Machine Learning Theory

Overview

Theoretical foundations of modern deep learning: universal approximation, optimisation landscape analysis, information theory, and the mathematics of attention mechanisms.

Why this matters

The gap between using ML tools and understanding what they compute. Required for serious research into AI systems.

Learning outcomes
  1. 1State and prove (simplified) Universal Approximation Theorem for neural networks.
  2. 2Analyse optimisation landscapes of neural networks (saddle points, overparameterisation).
  3. 3Understand information-theoretic foundations (entropy, mutual information, data processing inequality).
  4. 4Analyse attention mechanisms and transformers from a linear algebraic perspective.
  5. 5Apply concentration inequalities (Markov, Chebyshev, Hoeffding) to generalisation bounds.
  6. 6Understand the Neural Tangent Kernel and infinite-width networks.
⚡ Milestone checkpoint

Prove a two-layer network with sigmoid activations can approximate any continuous function on [0,1]^d. Derive the gradient of cross-entropy loss for a softmax classifier. Implement transformer attention from scratch in Python.

AI integration

Use Socratic mode for deep learning theory concepts. Ask AI to find flaws in your proof attempts for approximation theorems.
