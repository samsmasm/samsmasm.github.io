How to use Argument Mapper

  

    
Navigate
Drag blank canvas to pan · scroll to zoom · Ctrl Z to undo

    
Add nodes
Hover any node → click the coloured buttons that appear below it · or use + Node in the toolbar for full options

    
Drafts
+ Draft creates a floating node · click Connect → on it to attach to a parent, then pick its type

    
Change type
Click the type badge (Reason, Objection…) on any non-contention node to change its type

    
Detach
Hover a connector line · click the − button that appears to detach the child as an island

    
Co-premises
Drag two sibling nodes close together — they snap with a shared bar · click AND / OR to toggle the relationship · drag apart to separate

    
Collapse
Click ▾ on any node to hide its subtree · click ▸ to expand again

    
Tidy
Auto-arranges the full tree neatly · dissolves any co-premise groups

    
Save
Signed in: saves silently to My Maps · Guest: shows a 6-char code to share or bookmark · use Share in My Maps for a shareable link

    
Import
Copy the prompt, paste it with any text into an AI assistant, then paste the returned JSON to build a map automatically

    
Export
Generates a structured plain-text outline — copy to clipboard or download as .md / .txt

    
Check
Flags nodes with no evidence and objections with no rebuttal · summarises the top-level reasons for the contention

    
Print
Fits the whole map to the page and opens the print / PDF dialog
