# Diagrams and paragraphs: Checklist

| Key Thing to Check | Explanation | Student Response / Notes |
| :---- | :---- | :---- |
| **CRITERION A** | **GRAPHING & VISUAL DATA** |  |
| **Descriptive Title** | Does the diagram have a full title explaining the specific market (e.g., "The Market for Alcohol in Italy")? |  |
| **Non-Generic Axis Labels** | Have you avoided "P" and "Q" in favor of specific details like "Price of Petrol ($)"? |  |
| **Specific Units** | Are currencies (USD, EUR) or physical units (kg, liters) explicitly stated on the axes? |  |
| **Curve Labeling** | Is every curve clearly labeled (e.g., [![][image1]](https://www.codecogs.com/eqnedit.php?latex=D_1#0)[![][image2]](https://www.codecogs.com/eqnedit.php?latex=%2C%20#0)[![][image3]](https://www.codecogs.com/eqnedit.php?latex=D_2#0) or [![][image4]](https://www.codecogs.com/eqnedit.php?latex=S#0)[![][image5]](https://www.codecogs.com/eqnedit.php?latex=%2C%20#0)[![][image6]](https://www.codecogs.com/eqnedit.php?latex=S%2BTax#0)[![][image7]](https://www.codecogs.com/eqnedit.php?latex=\)%3F#0) |  |
| **Dynamic Elements** | Have you included arrows to show the direction of every shift or movement? |  |
| **Dotted Lines** | Are dotted lines used to project intersections to the exact points on the axes (e.g., [![][image8]](https://www.codecogs.com/eqnedit.php?latex=P_1#0), [![][image9]](https://www.codecogs.com/eqnedit.php?latex=Q_1#0))? |  |
| **Intersection Notation** | Are key points marked with letters (e.g., [![][image10]](https://www.codecogs.com/eqnedit.php?latex=E_1#0), [![][image11]](https://www.codecogs.com/eqnedit.php?latex=E_2#0)) for easy reference in your writing? |  |
| **Shaded Areas** | Have you shaded and labeled areas of significance, such as welfare loss or consumer surplus? |  |
| **Integrated Data** | If the article provides numbers (e.g., price rising from $10 to $15), are these marked on the axes? |  |
| **Presentation** | Is it large (1/4 page), neat, and in black ink for grayscale scanning legibility? |  |
| **CRITERIA B & C** | **TERMINOLOGY, APPLICATION & ANALYSIS** |  |
| **Direct Graph References** | Does the text explicitly refer to your labels (e.g., "the shift from [![][image12]](https://www.codecogs.com/eqnedit.php?latex=D_1#0) to [![][image13]](https://www.codecogs.com/eqnedit.php?latex=D_2#0) leads to [![][image14]](https://www.codecogs.com/eqnedit.php?latex=P_2#0)")? |  |
| **Logical Mechanism** | Have you explained *why* the curve shifted (the determinant) rather than just stating it moved? |  |
| **Step-by-Step Logic** | Is the explanation written for a non-expert, guiding them from initial state to final outcome? |  |
| **Stimulus Integration** | Have you used specific quantitative data or qualitative facts from the article to support claims? |  |
| **Paraphrasing** | Are references to the text paraphrased and integrated rather than long, uncritical quotes? |  |
| **Economic Terminology** | Have you used professional terms (e.g., "indirect tax") instead of "common" language? |  |
| **Key Concept Lens** | Is your chosen Key Concept (e.g., Intervention) referred to and used as a theme? |  |
| **Avoid Speculation** | Are all points strictly anchored in the article's evidence rather than "what if" scenarios? |  |
| **Bolding for Clarity** | Have you bolded your Key Concept and primary economic terms to assist the examiner? |  |
| **Word Count Check** | Are the paragraphs concise enough to fit within the 800-word limit? That is, is each paragraph about 175 words long each? |  |
