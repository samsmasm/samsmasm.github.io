What is this?

Numbers are arranged in a square spiral outward from the centre. Each faction takes turns claiming the lowest unclaimed number they can legally land on — a cell is blocked if an enemy faction's piece attacks it, but friendly pieces never block each other. The result is a surprisingly intricate, never-quite-repeating pattern that fills the whole grid.

The default setup (two factions, knight moves) is the Red & Blue Knights problem from Numberphile . Use the movement matrix to try bishops, rooks, or anything you invent.
