Spectra — How it works

Setting up a room

Write one statement per blank line. Put the category (or comma-separated categories) in square brackets at the end:

Homework should be banned. [Education]
AI will replace most jobs. [Technology, Ethics]

Set a scale (default 0–100) and optional endpoint labels (e.g. Never / Always). Toggle shuffle to randomise the order students see. Tick Require student names to collect names. Add a context note for your own reference during the session.

Sharing with students

Click QR to show the QR code and join link — students scan or type it and go straight to the sliders. No PIN needed. Each student gets an anonymous ID stored in their browser.

Display modes

  • Random — statements in their stored order, no grouping.
  • Organise — grouped by category with a coloured heading. A dashed summary bar at the bottom of each group shows the pooled average across all statements in that category.
  • Reveal — only category headings are shown (with live averages). Click a heading to expand or collapse the spectra beneath it.
  • Table — one row per student, one column per statement. Double-click a row to mark it for deletion, then click again to remove that student's responses.

Reading the spectra

Each coloured dot is one student's response. The dark vertical tick is the mean. Dots at the same position stack slightly so you can see clusters. Single-click a dot to see the student's name (if collected). Double-click a dot to mark it for deletion (it turns red ✕); click it again to remove just that response.

Other controls

  • Aa slider — adjusts the statement text size live.
  • Context — toggles your context note at the top of the results panel (amber banner). Greyed out if no context was set.
  • Leave — returns to the setup page. Responses are saved; use Rejoin to return.
